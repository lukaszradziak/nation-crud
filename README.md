# Nation CRUD

Simple CRUD App. Clients manager.

## How to run

`cp .env.example .env`

`touch database/database.sqlite`

`composer install`

`php artisan key:generate`

`php artisan migrate`

`php artisan db:seed`

`php artisan serve`

and go to `http://localhost:8000`
